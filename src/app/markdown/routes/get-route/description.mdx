## Get a route
`/api/routes/{id}`

<br/>

Get information about a specific route(s).

<br/>


## Parameters
| Name | Type | Description |
|------|------|-------------|
| route_id | string | Unique identifier for the route |
| start_point | string | Starting location of the route |
| end_point | string | Destination of the route |
| include_stops | boolean | Whether to include intermediate stops |
